# PromptKit

A comprehensive collection of production-ready prompts for common software development tasks, organized by category for quick access and implementation.

## Overview

PromptKit is a curated repository of detailed, actionable prompts designed to accelerate development workflows. Each prompt is crafted to generate complete, production-ready implementations with best practices baked in.

## Structure

Prompts are organized into the following categories:

- **authentication/** - User authentication flows, session management, password recovery
- **authorization/** - Role-based access control, permissions, security policies
- **frontend/** - React components, state management, UI/UX implementations
- **backend/** - API development, server logic, business layer implementations
- **infrastructure/** - DevOps, deployment, cloud configurations
- **testing/** - Unit tests, integration tests, E2E testing strategies
- **features/** - Common feature implementations (search, notifications, etc.)
- **integrations/** - Third-party service integrations (Google Maps, Stripe, etc.)
- **devops/** - CI/CD pipelines, deployment automation, monitoring
- **database/** - Schema design, migrations, query optimization

## How to Use

1. Browse the category folders to find the prompt you need
2. Copy the prompt content
3. Paste it into your AI assistant (Claude, ChatGPT, etc.)
4. Customize the variables/parameters as needed for your project
5. Review and integrate the generated code

## Prompt Features

Each prompt includes:
- Clear objective and scope
- Technology stack specifications
- Detailed implementation requirements
- Best practices and security considerations
- File structure expectations
- Testing requirements where applicable
